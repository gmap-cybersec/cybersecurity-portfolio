## 📋 Project Overview

Este projeto foi desenvolvido como parte de um curso de cibersegurança para praticar:

- ✅ Inventário e classificação de ativos
- ✅ Pontuação de risco e identificação de controles
- ✅ Checklist de conformidade (PCI DSS & GDPR)
- ✅ Mapeamento de controles por categoria (Administrativo, Técnico, Físico)

## ✅ Controls Mapped by Category

| Control Name | Type | Purpose |
|---|---|---|
| **Firewall** | Technical / Preventative | Bloquear tráfego malicioso |
| **Password Policy** | Managerial / Preventative | Prevenir ataques de força bruta |
| **IDS** | Technical / Detective | Detectar comportamentos suspeitos |
| **CCTV** | Physical / Preventative & Detective | Prevenir acesso físico não autorizado |
